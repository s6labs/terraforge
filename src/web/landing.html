Capabilities

Everything your AI stack needs

Six core pillars that turn a collection of personal devices into production-grade AI infrastructure.

⚡
Opportunistic GPU Pool
Idle GPU cycles across every node are aggregated into a shared compute pool. VRAM-aware routing dispatches jobs to the best available hardware automatically — no Kubernetes required.
🔒
Secure Mesh Network
Headscale + WireGuard (ChaCha20Poly1305) forms an encrypted private network. No node ever exposes a public port. All traffic stays inside your tailnet — MagicDNS included.
🤖
Agent Hosting Layer
OpenFang and OpenClaw run 24/7 on the best available node, automatically migrating when hardware goes offline or load spikes. Deploy from the dashboard in one click.
🖥️
Developer Workspaces
Coder workspaces provisioned on any node from your browser in under 60 seconds. GPU-backed templates generated by the TerraForge LLM engine — including the one you're reading now.
🧠
Multi-Provider LLM Router
Ollama, Anthropic, OpenRouter, LM Studio — all unified behind a single OpenAI-compatible endpoint. Local-first with automatic cloud fallback. Zero config changes to switch providers.
☁️
Cloud GPU Fallback
When local resources are exhausted, GCP spot instances auto-provision via Terraform, join the network, execute the job, and self-destruct after 15 minutes idle. Opt-in with cost preview.
System Design

Four layers. Zero dependencies on each other.

Each layer has a clean API contract. Independently deployable, independently testable.

Headscale WireGuard Tailscale

Coordinator
GCP Free Tier — $0 / month
e2-micro (2 vCPU · 1 GB RAM · 30 GB disk). Runs Headscale + TerraForge Server in Docker. Coordination-only — all compute lives on your nodes.
Security
WireGuard everywhere
ChaCha20Poly1305 on every packet. No node exposes a public port. Coordinator surface: port 443 only. Dashboard is tailnet-only — unreachable from the open internet.
Scheduling
VRAM-aware priority routing
40% GPU availability · 30% load · 20% latency · 10% preference. Exponential backoff retries. Cloud fallback triggers only when the entire local pool is exhausted.
Node Join Flow

Any device joins in under 5 minutes.

One command. OS auto-detected. No manual networking configuration required.

01
Run the Join Script
curl .../join | bash

OS detected automatically. Tailscale client installed silently in the background.
02
Pre-Auth Key Issued
The script calls the TerraForge API. A single-use Headscale pre-auth key is generated and expires after 1 hour.
03
VPN Tunnel Up
Node receives a private 100.64.x.x address. WireGuard tunnel established. Reachable by hostname via MagicDNS.
04
Live in Dashboard
TerraForge Agent starts as a system service, reports hardware, and appears in the dashboard — GPU bars and all — within seconds.
GPU Pool

Every idle cycle — pooled.

nvidia-smi, rocm-smi, and Apple Silicon unified memory are detected, normalized, and made available as a shared resource.

VRAM-aware routing: 7B ≈ 6 GB · 13B ≈ 10 GB · 70B ≈ 40 GB at 4-bit
